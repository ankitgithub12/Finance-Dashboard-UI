# RBAC Finance Dashboard Client

This application serves as the frontend client for the Finance Dashboard. It is built using React, Vite, Tailwind CSS, and Recharts.

For full documentation regarding features, architecture, and deployment setup, please consult the primary [README](../README.md) located at the root of the project source code.

## Running Locally

To run the Vite development server:
```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - Starts the development server.
- `npm run build` - Builds the app for production into the `dist` folder.
- `npm run lint` - Runs ESLint to check for code quality issues.
- `npm run preview` - Locally preview the production build.
